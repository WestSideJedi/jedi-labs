## Scripts

### decode64.py
As simple Python Script that decodes Base64-encoded strings. Useful for quick decoded during CTFs, HTB, and red team assessments


Usage:
$ python decode.py
Enter Base64-encoded string: SGVsbG8gV29ybGQh
Decoded Sting: Hello World!
